Execute Code

Executes a snippet of code in a sandboxed environment. The input data is available within the script's scope, and the script's return value becomes the action's output.

Overview

The Execute Code action brings the full power of programming languages into your workflows. Write custom JavaScript code to perform complex calculations, data manipulations, business logic, or any custom processing that can't be achieved with standard actions. Perfect for custom algorithms, data validation, complex transformations, and integrating with external libraries.

Data Validation

Validate user data and return validation results

JavaScript Code:

const user = input.user;
const errors = [];

if (!user.email || !user.email.includes('@')) {
  errors.push('Invalid email address');
}

if (!user.age || user.age < 18) {
  errors.push('Must be 18 or older');
}

return {
  isValid: errors.length === 0,
  errors: errors,
  user: user
};

Array Processing

Process an array of items and calculate statistics

JavaScript Code:

const items = input.items || [];

const total = items.reduce((sum, item) => sum + item.price, 0);
const average = items.length > 0 ? total / items.length : 0;
const maxPrice = Math.max(...items.map(item => item.price));
const minPrice = Math.min(...items.map(item => item.price));

return {
  count: items.length,
  total: total,
  average: Math.round(average * 100) / 100,
  maxPrice: maxPrice,
  minPrice: minPrice,
  expensiveItems: items.filter(item => item.price > average)
};

String Processing

Clean and format text data

JavaScript Code:

const text = input.text || '';

// Clean and format the text
const cleaned = text
  .trim()
  .toLowerCase()
  .replace(/[^a-z0-9\s]/g, '')
  .replace(/\s+/g, ' ');

const words = cleaned.split(' ').filter(word => word.length > 0);
const wordCount = words.length;
const uniqueWords = [...new Set(words)];

return {
  original: text,
  cleaned: cleaned,
  wordCount: wordCount,
  uniqueWordCount: uniqueWords.length,
  words: words,
  uniqueWords: uniqueWords
};

Configuration Schema

Configure the Execute Code action with the programming language and code to execute. 

{
  "title": "Execute Code Configuration",
  "type": "object",
  "properties": {
    "language": {
      "type": "string",
      "enum": ["javascript"],
      "default": "javascript",
      "description": "The programming language of the code snippet."
    },
    "code": {
      "type": "string",
      "title": "Code Snippet",
      "description": "The code to execute. The input data is available as a variable named 'input'."
    }
  },
  "required": ["language", "code"]
}
