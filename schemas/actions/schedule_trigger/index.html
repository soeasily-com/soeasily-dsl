Schedule Trigger

Starts a workflow automatically based on a defined schedule. Provide a standard CRON expression for precise, recurring execution.

Overview

The Schedule Trigger enables time-based workflow automation using CRON expressions. Whether you need to run reports daily, send reminders weekly, or perform maintenance tasks monthly, this trigger provides precise scheduling capabilities. Perfect for recurring tasks, batch processing, and automated maintenance workflows.

CRON Expression Examples

Common CRON patterns for different scheduling needs. CRON format: minute hour day month day-of-week

Every Hour

Runs at the beginning of every hour (e.g., 1:00, 2:00, 3:00)

0 * * * *

Daily at 9:00 AM

Runs once per day at 9:00 AM

0 9 * * *

Weekdays at 5:30 PM

Runs Monday through Friday at 5:30 PM

30 17 * * 1-5

Weekly on Sundays at Midnight

Runs every Sunday at 12:00 AM

0 0 * * 0

Monthly on the 1st at 6:00 AM

Runs on the first day of every month at 6:00 AM

0 6 1 * *

Every 15 Minutes

Runs every 15 minutes (e.g., :00, :15, :30, :45)

*/15 * * * *

Configuration Schema

Configure the Schedule Trigger with a CRON expression that defines when the workflow should run. 

{
  "title": "Schedule Configuration",
  "type": "object",
  "properties": {
    "cron_expression": {
      "title": "CRON Expression",
      "description": "A standard CRON expression to define the schedule (e.g., '0 9 * * MON-FRI').",
      "type": "string",
      "examples": ["0 * * * *"]
    }
  },
  "required": ["cron_expression"]
}

Input & Output

Input Schema

This trigger does not accept input from other actions as it's designed to start workflows.

{
  "title": "Schedule Trigger Input",
  "description": "This trigger does not accept input from other actions.",
  "type": "object",
  "properties": {},
  "additionalProperties": false
}

Output Schema

Provides the timestamp of when the workflow was triggered.

{
  "title": "Schedule Trigger Output",
  "type": "object",
  "properties": {
    "triggered_at": {
      "description": "The ISO 8601 timestamp of the trigger execution time.",
      "type": "string",
      "format": "date-time"
    }
  },
  "required": ["triggered_at"]
}
