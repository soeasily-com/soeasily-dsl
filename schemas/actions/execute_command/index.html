Execute Command

Executes a shell command in the execution environment. Captures stdout, stderr, and the exit code for comprehensive command execution control.

Overview

The Execute Command action brings the full power of the command line to your workflows. Execute shell commands, run scripts, interact with system utilities, and integrate with command-line tools. Perfect for system administration tasks, file operations, data processing with CLI tools, and integrating with existing shell-based automation.

Common Command Patterns

Useful command patterns for common workflow tasks.

File Operations

# List files
ls -la /path/to/directory

# Copy files
cp source.txt destination.txt

# Move/rename files
mv oldname.txt newname.txt

# Create directories
mkdir -p /path/to/new/directory

# Remove files/directories
rm -rf /path/to/remove

Text Processing

# Search in files
grep "pattern" file.txt

# Count lines/words
wc -l file.txt

# Sort and unique
sort file.txt | uniq

# Replace text
sed 's/old/new/g' file.txt

# Extract columns
cut -d',' -f1,3 data.csv

Data Processing

# JSON processing
jq '.field' data.json

# CSV processing
csvcut -c 1,3 data.csv

# XML processing
xmllint --format data.xml

# Base64 encoding/decoding
echo "text" | base64
echo "dGV4dA==" | base64 -d

Network Operations

# Download files
curl -O https://example.com/file.txt

# HTTP requests
curl -X POST -H "Content-Type: application/json" \
  -d '{"key":"value"}' https://api.example.com

# Check connectivity
ping -c 3 example.com

# DNS lookup
nslookup example.com

System Information

# System info
uname -a

# Disk usage
df -h

# Memory usage
free -h

# Process list
ps aux

# Environment variables
env | grep PATH

Archive Operations

# Create tar archive
tar -czf archive.tar.gz directory/

# Extract tar archive
tar -xzf archive.tar.gz

# Create zip archive
zip -r archive.zip directory/

# Extract zip archive
unzip archive.zip
