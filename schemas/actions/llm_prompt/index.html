LLM Prompt

Sends a prompt to a specified Large Language Model and returns the generated text.

Overview

The LLM Prompt action brings the power of Large Language Models directly into your workflows. Connect to models like GPT-4, Claude, or other LLMs to generate text, analyze content, answer questions, or perform complex reasoning tasks. Perfect for content generation, data analysis, customer support automation, and intelligent decision-making.

Configuration Schema

Configure the LLM Prompt with model selection, prompt text, and generation parameters. 

{
  "title": "LLM Prompt Configuration",
  "type": "object",
  "properties": {
    "model": { 
      "type": "string", 
      "description": "The identifier of the LLM to use (e.g., 'gpt-4-turbo')." 
    },
    "prompt": { 
      "type": "string", 
      "description": "The prompt to send to the model." 
    },
    "temperature": { 
      "type": "number", 
      "minimum": 0, 
      "maximum": 2, 
      "default": 0.7 
    }
  },
  "required": ["model", "prompt"]
}

Input & Output

Input Schema

Accepts any JSON object that can be used to dynamically populate the prompt.

{
  "title": "LLM Prompt Input",
  "type": "object"
}

Output Schema

Returns the generated text from the language model.

{
  "title": "LLM Prompt Output",
  "type": "object",
  "properties": {
    "generated_text": { "type": "string" }
  },
  "required": ["generated_text"]
}

Test Examples

Here are example configurations and expected outputs to help you understand how the LLM Prompt action works.

Ask a Simple Question

Tests generating text from a simple prompt.

Configuration:

{
  "model": "gpt-4-turbo",
  "prompt": "What is the capital of France?"
}

Expected Output:

{
  "generated_text": "The capital of France is Paris."
}

Content Generation with Temperature

Generate creative content with higher temperature for more creativity.

Configuration:

{
  "model": "gpt-4-turbo",
  "prompt": "Write a creative tagline for a workflow automation platform",
  "temperature": 1.2
}

Example Output:

{
  "generated_text": "Transform chaos into choreography with intelligent automation that thinks ahead."
}

Data Analysis Prompt

Use AI to analyze and summarize data or content.

Configuration:

{
  "model": "gpt-4-turbo",
  "prompt": "Analyze this customer feedback and provide a sentiment score (1-10) and key themes: 'The product is amazing but the support response time could be better. Love the features!'",
  "temperature": 0.3
}

Example Output:

{
  "generated_text": "Sentiment Score: 7/10\n\nKey Themes:\n- Product satisfaction: High (positive feedback on features)\n- Support concerns: Moderate (response time issues)\n- Overall experience: Positive with room for improvement\n\nRecommendation: Focus on improving support response times while maintaining product quality."
}
