Debug Log

Logs the input data to the workflow's execution history with a custom message. This action is essential for debugging and passes data through without modification.

Overview

The Debug Log action is your essential debugging companion for workflow development and troubleshooting. It captures snapshots of your data at any point in the workflow execution, logs them with custom messages, and passes the data through unchanged. Perfect for understanding data flow, diagnosing issues, monitoring workflow execution, and validating transformations.

Configuration Schema

Configure the Debug Log action with a descriptive message for the log entry. 

{
  "title": "Debug Log Configuration",
  "type": "object",
  "properties": {
    "message": {
      "type": "string",
      "description": "A message to include with the log entry. Can contain expressions."
    }
  },
  "required": ["message"]
}
