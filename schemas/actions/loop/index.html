Loop Data

Iterates over an array from the input data. The connected sequence of actions will be executed for each item in the array.

Control Flow

Overview

Action ID

soeasily/loop

Version

1.0.0

Handler

soeasily.control.loop@1

Tags

loop, for-each, iterate, array, list

Configuration

Required Configuration

Property Type Required Description
array_path string Yes A JSONata expression pointing to the array in the input data to iterate over.

Example Configuration:

{ "array_path": "source.body.line_items" }

Input Schema

Expected Input

An object containing the array to be looped over. The structure depends on the array_path configuration.

Example Input:

{ "users": [ { "id": "usr_1", "name": "Alice" }, { "id": "usr_2", "name": "Bob" }, { "id": "usr_3", "name": "Charlie" } ] }

Output Schema

Output Per Iteration

For each iteration, this action outputs an object containing the current item's value and its index.

Property Type Required Description
item any Yes The current item from the source array.
index integer Yes The zero-based index of the current item.

Example Output (First Iteration):

{ "item": { "id": "usr_1", "name": "Alice" }, "index": 0 }
