HTTP Request

Makes a customizable HTTP request to a specified URL and returns the full response, including status code, headers, and body.

Overview

The HTTP Request action is your gateway to integrating with external APIs and web services. It supports all major HTTP methods (GET, POST, PUT, PATCH, DELETE) and provides complete control over headers, query parameters, and request bodies. Perfect for REST API integrations, webhook calls, and data synchronization.

Configuration Schema

Configure the HTTP Request with method, URL, headers, query parameters, and request body. 

{
  "title": "HTTP Request Configuration",
  "type": "object",
  "properties": {
    "method": {
      "type": "string",
      "enum": ["GET", "POST", "PUT", "PATCH", "DELETE"],
      "default": "GET"
    },
    "url": { 
      "type": "string", 
      "format": "uri" 
    },
    "headers": { 
      "type": "object" 
    },
    "query_params": { 
      "type": "object" 
    },
    "body": { 
      "type": ["object", "string"] 
    }
  },
  "required": ["method", "url"]
}

Input & Output

Input Schema

Accepts any JSON object that can be used to dynamically populate request parameters.

{
  "title": "HTTP Request Input",
  "type": "object"

Output Schema

Returns the complete HTTP response with status code, headers, and body.

{
  "title": "HTTP Request Output",
  "type": "object",
  "properties": {
    "status_code": { "type": "integer" },
    "headers": { "type": "object" },
    "body": {}
  },
  "required": ["status_code", "headers", "body"]
}

Test Examples

Here are example configurations and expected outputs to help you understand how the HTTP Request action works.

GET Request to JSONPlaceholder

Tests a simple GET request to a public API.

Configuration:

{
  "method": "GET",
  "url": "https://jsonplaceholder.typicode.com/todos/1"
}

Expected Output:

{
  "status_code": 200,
  "headers": { 
    "content-type": "application/json; charset=utf-8" 
  },
  "body": {
    "userId": 1,
    "id": 1,
    "title": "delectus aut autem",
    "completed": false
  }
}

POST Request with JSON Body

Example of sending data to an API endpoint.

Configuration:

{
  "method": "POST",
  "url": "https://api.example.com/users",
  "headers": {
    "Content-Type": "application/json",
    "Authorization": "Bearer token123"
  },
  "body": {
    "name": "John Doe",
    "email": "john@example.com"
  }
}

GET Request with Query Parameters

Example of adding query parameters to a request.

Configuration:

{
  "method": "GET",
  "url": "https://api.example.com/search",
  "query_params": {
    "q": "workflow automation",
    "limit": 10,
    "sort": "date"
  },
  "headers": {
    "Accept": "application/json"
  }
}
