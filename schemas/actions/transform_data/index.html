Transform Data

Reshapes the input data into a completely new structure using a JSONata expression. Ideal for formatting data before sending it to an external API.

Overview

The Transform Data action is your data reshaping powerhouse. Using JSONata expressions, it can completely restructure, filter, aggregate, and format your data to match any required output format. Whether you're preparing data for an API call, creating reports, or normalizing data from different sources, this action provides the flexibility you need.

Order Summary

Calculate totals and format order data

Input:

{
  "order_id": "ord-456",
  "items": [
    {"name": "Widget A", "price": 10, "qty": 2},
    {"name": "Widget B", "price": 15, "qty": 1}
  ]
}
â†’

Output:

{
  "order": "ord-456",
  "total": 35,
  "item_count": 3,
  "summary": "2x Widget A, 1x Widget B"
}

JSONata Expression:

{
  "order": order_id,
  "total": $sum(items.(price * qty)),
  "item_count": $sum(items.qty),
  "summary": $join(items.(qty & "x " & name), ", ")
}

Array Filtering & Mapping

Filter and transform array data

Input:

{
  "users": [
    {"name": "Alice", "age": 25, "active": true},
    {"name": "Bob", "age": 17, "active": false},
    {"name": "Carol", "age": 30, "active": true}
  ]
}
â†’

Output:

{
  "active_adults": [
    {"username": "Alice", "category": "adult"},
    {"username": "Carol", "category": "adult"}
  ]
}

JSONata Expression:

{
  "active_adults": users[active = true and age >= 18].{
    "username": name,
    "category": "adult"
  }
}

JSONata Quick Reference

Common JSONata patterns for data transformation.

Basic Operations

// Field access
user.name

// Array access
items[0]

// Conditional
age >= 18 ? "adult" : "minor"

Array Functions

// Filter
items[price > 10]

// Map
items.name

// Sum
$sum(items.price)

String Functions

// Concatenate
$join(items.name, ", ")

// Uppercase
$uppercase(name)

// Substring
$substring(text, 0, 10)

Object Construction

// New object
{
  "id": user_id,
  "full_name": first & " " & last
}

// Merge objects
user ~> {"updated": $now()}
