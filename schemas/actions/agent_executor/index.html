Agent Executor

Executes a sequence of actions driven by a Large Language Model to achieve a high-level objective. The agent has access to a defined set of tools (other SoEasily actions).

Overview

The Agent Executor represents the cutting edge of AI-powered workflow automation. By combining Large Language Models with a curated set of tools, it can autonomously plan, reason, and execute complex multi-step tasks. Unlike traditional workflows that follow predefined paths, the Agent Executor dynamically determines the best sequence of actions to achieve your objectives, making it perfect for complex problem-solving, research tasks, and adaptive automation scenarios.

How Agent Executor Works

The Agent Executor follows a reasoning loop where it analyzes the objective, plans actions, executes tools, and evaluates results until the goal is achieved.

1. Analyze Objective ‚Üí Understand the high-level goal
2. Plan Actions ‚Üí Determine which tools to use and in what order
3. Execute Tool ‚Üí Run the selected SoEasily action
4. Evaluate Result ‚Üí Assess if the goal is achieved or if more steps are needed
5. Repeat or Conclude ‚Üí Continue the loop or provide final answer
